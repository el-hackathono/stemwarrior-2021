What is Phishing?

Phishing is duplicating the appearence of a website such as a Facebook Login Page.
The using this unsuspected appearence to trick people into inputing personal account information
in the website. It is a felony if done without gaining prior consent, and notifying the other person about the link.

How is Phishing Done?

Phishing is normally done by copying and pasting the source code of a website, such as Facebook from the previous example.
Using that to show a similar appearence to the standard website, and then slightly altering the code,
to send the password and email information entered to a python or php script, granting the owner of the fake webpage, access to your personal information.
